# Investment Log Tistory Skin

티스토리 스킨 업로드용 폴더입니다.

구성:
- `index.xml`
- `skin.html`
- `style.css`

업로드 방법:
1. 이 폴더의 파일들을 ZIP으로 압축합니다.
2. 티스토리 관리자 > 꾸미기 > 스킨 변경 > 스킨 등록에서 ZIP을 업로드합니다.
3. 업로드 후 스킨을 적용합니다.

메모:
- 티스토리 공식 스킨 가이드의 파일 구조와 치환자 구조를 기준으로 작성했습니다.
- 필수 파일 구조는 공식 문서의 `index.xml`, `skin.html`, `style.css`를 따릅니다.
- 참고: https://tistory.github.io/document-tistory-skin/common/files.html
- 참고: https://tistory.github.io/document-tistory-skin/common/index.xml.html
- 참고: https://tistory.github.io/document-tistory-skin/common/global.html

주의:
- 미리보기 이미지 파일은 포함하지 않았습니다. 업로드 자체에는 필수는 아니지만, 필요하면 `preview256.jpg`, `preview560.jpg`, `preview1600.jpg`를 추가할 수 있습니다.
- 티스토리 본문 HTML 구조와 댓글/방명록 렌더링은 플랫폼 정책에 따라 일부 DOM이 달라질 수 있습니다.
