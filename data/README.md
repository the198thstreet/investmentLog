# Data Placeholder

이 폴더는 향후 보조 JSON 산출물을 연결하기 위한 자리입니다.

- `home-summary.json`
- `reports/index.json`
- `tickers/index.json`
- `tickers/{code}.json`

정적 페이지는 현재 더미 데이터로 작성되어 있으며, 반복 블록 클래스와 HTML 주석 위치를 기준으로 Jinja2 또는 JSON 바인딩으로 전환할 수 있습니다.

현재 일별 게시물 샘플은 아래 파일 두 개입니다.

- `reports/2026-03-27.html`
- `reports/2026-03-28.html`

`stockTracker`는 위 두 파일 중 하나를 샘플 템플릿 구조로 참고해, 매일 같은 형식의 HTML 파일을 추가 생성하면 됩니다.
